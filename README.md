# Getting Started with ESP8266 ESP-01 fresh chip 

In this guide I show how to check ESP 01 `AT` command with Ardunio
## Component we need
- ESP8266 ESP-01 Wi-Fi Module
- Male/Female Jumper Wire
- Breadboard
- Ardunio Uno R3
